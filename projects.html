Professional Certificate in Coding: Full Stack Development with MERN - Feb 2023

Here are a few of the Projects we've been working on throughout the course:


Project Title: PacMen Exercise



Project Description: In the PacMen exercise we were required to create a webpage where we can add PacMen with random positioning and velocity with the click of a button. We were to achieve this by influencing the movement of the PacMen by manipulating the DOM.

PacMen Exercise Repository



Project Title: Eye Exercise



Project Description: In the Eye exercise we were given starter required to Create an additional eye that tracks the movement of the mouse on the webpage. The starter code for one eye was already given and was to be modified to have 2 eyes.

Eye Exercise Repository



Project Title: Real Time Bus Tracker



Project Description: In the Real Time Bus Tracker exercise we were required to animate the bus route from MIT to Harvard by manipulating the DOM and display the animations on a map. MapBox was used to get the map reference and the GPS coordinates for the bus stops along the way were also provided.

Real Time Bus Tracker Exercise Repository
